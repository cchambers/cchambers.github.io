Adapt Content

Usage

This component allows you to perform and manage element level media queries... for example, if you needed the text to change when the sh-adapt-content container was 400px wide:

      
        <sh-adapt-content>
          default content
          <div slot="sm">
            small
          </div>
          <div slot="md">
            medium
          </div>
          <div slot="lg">
            large
          </div>
          <div slot="xl">
            xtra-large
          </div>
        </sh-adapt-content>

Reference

Attributes

Attribute Type Description
hidden Boolean Hide from view.

Slots

Slot Description
default Text content.
sm HTML content of banner at sm breakpoint.
md HTML content of banner at md breakpoint.
lg HTML content of banner at lg breakpoint.
xl HTML content of banner at xl breakpoint.
