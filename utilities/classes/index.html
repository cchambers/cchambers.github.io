OO Classes

Usage

OO classes that can be used to style or space content. They are not used in component creation, but the variables used to create them are. 

      
        <p class="px30 font-primary" >Lorem ipsum dolor...</p>

Box Model

Padding

Class Effect
.pad{-edge}{-amount}
  • Modify the padding of an element.
  • Left+right edge combined is x, top+bottom edges combined is y. Other options are "t", "r", "b", "l".
  • Options for 'amount' are: none, 'quarter', 'half', and 'double'.

Margin

Class Effect
.margin{-edge}{-amount}
  • Modify the margin of an element. Left+right edge combined is x, top+bottom edges combined is y.
  • Left+right edge combined is x, top+bottom edges combined is y. Other options are "t", "r", "b", "l".
  • Options for 'amount' are: none, 'quarter', 'half', and 'double'.
.pull{-edge}{-amount}
  • Modify the negative margin of an element.
  • Left+right edge combined is x, top+bottom edges combined is y. Other options are "t", "r", "b", "l".
  • Options for 'amount' are: none, 'quarter', 'half', and 'double'.

Typography

Size and Line Height

Class Effect
.pxNN
  • Set the [responsive] font size of a piece of content.
  • Options range from 10-100. ex: .px62
.lhNN
  • Set the [responsive] line height of a piece of content.
  • Options range from 10-100. ex: .lh30

Alignment

Class Effect
.text-right Right align text.
.text-left Left align text.
.text-center Center align text.
.text-justify Align text justified.
.force-justify Align text justified forced (justifies the last line and single-line paragraphs).

Casing

Class Effect
.uppercase
Uppercased text.
.lowercase
Lowercased text.
.normalcase
Normal casing.
.titlecase
Title Casing.
